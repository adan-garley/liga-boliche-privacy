Política de Privacidad

Liga de Boliche

Fecha de entrada en vigor: 22-06-2025
Última actualización: 22-06-2025
Resumen: Liga de Boliche no recopila, almacena ni comparte ningún dato personal. Todos los datos de tu juego se guardan únicamente en tu dispositivo.

1. Información que Recopilamos

Liga de Boliche NO recopila ningún tipo de información personal. La aplicación funciona completamente de manera local en tu dispositivo.

Datos que NO recopilamos:

  • Información personal (nombre, email, teléfono)
  • Datos de ubicación
  • Información del dispositivo
  • Datos de uso o estadísticas
  • Cookies o identificadores únicos
  • Información de contactos

2. Almacenamiento de Datos del Juego

Todos los datos relacionados con tus partidas de boliche (puntuaciones, handicaps, historial) se almacenan exclusivamente en tu dispositivo mediante la base de datos local de la aplicación.

Características del almacenamiento local:

  • Los datos nunca salen de tu dispositivo
  • No se envían a servidores externos
  • Solo tú tienes acceso a esta información
  • Los datos se eliminan si desinstalas la aplicación

3. Permisos de la Aplicación

Liga de Boliche no solicita permisos especiales en tu dispositivo. La aplicación no necesita acceso a:

  • Internet o conexión de red
  • Cámara o micrófono
  • Ubicación GPS
  • Contactos o calendario
  • Almacenamiento externo

4. Compartir Información

Como no recopilamos ningún dato personal, no compartimos, vendemos, alquilamos o distribuimos información a terceros, anunciantes, o cualquier otra entidad.

5. Seguridad de los Datos

Aunque no manejamos datos personales, implementamos las mejores prácticas de seguridad para proteger los datos locales de tu juego:

  • Almacenamiento seguro en la base de datos local del dispositivo
  • Sin transmisión de datos por internet
  • Código desarrollado siguiendo estándares de seguridad

6. Servicios de Terceros

Liga de Boliche no utiliza servicios de terceros que puedan recopilar información, incluyendo:

  • Servicios de analíticas
  • Redes publicitarias
  • Servicios de crash reporting
  • Servicios de autenticación

7. Menores de Edad

Liga de Boliche es segura para usuarios de todas las edades. Como no recopilamos ningún tipo de información personal, no hay riesgo de recopilación inadvertida de datos de menores de 13 años.

8. Cambios en esta Política

Si realizamos cambios significativos en esta Política de Privacidad, notificaremos a los usuarios a través de una actualización de la aplicación. El uso continuado de la aplicación después de dichos cambios constituye la aceptación de la nueva política.

9. Derechos del Usuario

Como no recopilamos datos personales, no es necesario solicitar:

  • Acceso a datos personales
  • Rectificación o eliminación de datos
  • Portabilidad de datos

Sin embargo, puedes eliminar todos los datos de la aplicación desinstalándola de tu dispositivo.

10. Jurisdicción

Esta Política de Privacidad se rige por las leyes aplicables de protección de datos y privacidad. Al ser una aplicación que no recopila datos, cumple con regulaciones como GDPR, CCPA y otras normativas de privacidad.
